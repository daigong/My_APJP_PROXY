PRE-INSTALLATION
----------------
- create HEROKU account (http://devcenter.heroku.com/articles/quickstart)
- install HEROKU SDK (http://devcenter.heroku.com/articles/quickstart)

INSTALLATION
------------
- open src\main\resources\APJP.properties
	- configure
		- APJP_KEY == generated key
		- APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_1_KEY
		- APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_1_VALUE
		- APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_2_KEY
		- APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_2_VALUE
		- APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_3_KEY
		- APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_3_VALUE
		- APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_4_KEY
		- APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_4_VALUE
		- APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_5_KEY
		- APJP_REMOTE_HTTP_SERVER_RESPONSE_PROPERTY_5_VALUE
		- APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_1_KEY
		- APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_1_VALUE
		- APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_2_KEY
		- APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_2_VALUE
		- APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_3_KEY
		- APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_3_VALUE
		- APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_4_KEY
		- APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_4_VALUE
		- APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_5_KEY
		- APJP_REMOTE_HTTPS_SERVER_RESPONSE_PROPERTY_5_VALUE
- install APJP FOR HEROKU
	- open CONSOLE
		- heroku login
		- git init
		- git add .
		- git commit -m "init"
		- heroku create --stack cedar
		- git push heroku master
		- heroku open

IMPORTANT
---------
- APJP_REMOTE_HTTP_SERVER_REQUEST_URL == http://application-id.herokuapp.com/HTTP
- APJP_REMOTE_HTTPS_SERVER_REQUEST_URL == http://application-id.herokuapp.com/HTTPS
